# Skills

Project-specific skill sources live in this folder. `web/` React is the canonical frontend; deprecated Streamlit remnants are not supported by these skills.

High-level routing:

- Use `senior-software-engineer-agent` for architecture and cross-layer planning.
- Use `review-ui-api-integrations` for full-stack code review.
- Use `feature-contracts-specs` to define an API contract.
- Use `cross-stack-contract-parity` to verify the implemented React and FastAPI contract.
- Use `web-ux-consistency-agent` for React UI design and review.
- Use the focused backend, migration, testing, requirements, Bruno, and workflow skills for their named tasks.
